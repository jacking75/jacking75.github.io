### tar.bz2 파일 압축 풀기

```
tar -xvf test.tar.bz2
```
  
  
tar.bz2 압축 내용 보기  
```
tar -tvf test.tar.bz2
```
  
  
  
### tar.gz 파일 압축 풀기
  
```
tar -xvzf test.tar.gz
```
