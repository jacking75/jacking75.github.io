```
Test.dsdsds = 10;
Test.ds = 10;
```  
   
이런 코드가 있을 때 아래처럼 = 중심으로 코드를 정렬하고 싶을 때가 있을 것이다.  
```
Test.dsdsds	= 10;
Test.ds		= 10;
```  
  
<br>  
    
이런 것을 Visual Studio의 확장 툴인 ‘Code alignment’을 설치하면 한방에 깔끔하게 해준다.  
공식 사이트는 http://www.codealignment.com/ 이다.  
그리고 Notepad++도 지원한다.  
   
사용 방법은 아래의 동영상이나 그림을 참고하면 된다.
